# Convertidor TXT a Markdown (bloque de código)

Aplicación gráfica con ttkbootstrap que convierte cualquier texto (pegado o desde un archivo .txt) en un bloque de código Markdown ```markdown ... ``` sin modificar el contenido original.

## Requisitos
- Python 3.8+
- ttkbootstrap

## Instalación
```bash
pip install ttkbootstrap
