# 📄 TXT to Markdown Converter

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20Windows%20%7C%20macOS-lightgrey)

Aplicación de escritorio con interfaz gráfica moderna (estilo iOS) que convierte cualquier texto o archivo `.txt` en un bloque de código **Markdown** (` ```markdown ... ``` `) sin modificar ni una sola línea del contenido original. Ideal para documentación, ejemplos de código o preparar texto para publicar en plataformas que usan Markdown.

## 🚀 Características

- 🖱️ **Dos modos de entrada**: pegar texto directamente o cargar un archivo `.txt`.
- 🎨 **Interfaz moderna** con `ttkbootstrap` (tema iOS claro/oscuro).
- 📦 **Conversión sin pérdidas** – respeta espacios, saltos de línea, caracteres especiales y formato original.
- 💾 **Guardado como archivo `.md`** listo para usar en GitHub, GitLab, blogs, etc.
- 🔒 **No modifica el original** – crea un nuevo archivo envuelto en bloque de código.

## 📋 Requisitos

- Python 3.8 o superior
- pip (gestor de paquetes de Python)
